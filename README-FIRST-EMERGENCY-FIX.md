# Emergency Root-Level Website Fix

This package is designed to repair the website immediately even if GitHub folder uploads are causing problems.

Upload every file in this folder directly to the ROOT of the GitHub repository `Pjudge7/patrickljudge-site`.

Required root-level files:
- index.html
- robots.txt
- sitemap.xml
- CNAME
- patrick-l-judge-business-card-final.png
- patrick-judge-digital-business-card.png
- patrick-l-judge-qr.png
- patrickljudge-website-qr.png
- patrick-l-judge-contact.vcf
- patrick-judge-contact.vcf
